Click on following button to download the library. This library is .aar file. You need to include this library inside your android project.

Generate API Key

Please sign up on our platform in order to register yourself as developer. After signing up, please create the profile for your application using following link.

If your Application already exists on our platform, then contact us to claim your application.


After your account has been created, you will be able to see API key in your account.

Configure Library

You need to configure this API key in your resources folder. Generally, you need to configure inside strings.xml file which is located in app / res / values / strings.xml

After configuring API key, you can use our tutorial view in side your activity using our custom view.

How to Import a AAR File:
  1. Go to File
  2. Click on New
  3. Click on New Module
  4. Select Import Library AAR/JAR
  5. Locate the Library
  6. Click on Finish.

More info available at: http://stackoverflow.com/questions/24506648/addinglocalaarfilestogradlebuildusingflatdirsisnotworking

How to include a offline Tippie:
  1. Make assets " folder inside " main" .
  2. Inside, assets folder, Make tippies " folder.
  3. Inside tippies folder, copy the .tips file provided by us.
    4. Note: name of .tips file is its unique identifier( of type long). e.g. 1234456.tips. So, do not change names of .tips file provided by us.

Procedures:
1. Initialize SDK

in onCreate() function before launching any tutorial write following code.

TippieSDK.initialize(this);
2. Launch a Tippie
Use this method to directly launch a Tippie. 
TippieSDK.launchTippie (l ong tippieId ,String languagePack ,int tippieType ,booleanisOnline,int requestId,Context context);
 e.g.TippieSDK.launchTippie(1456610546, "em", TippieSDK.TIPPIE_INTRO, false,ONBOARDING_REQUEST_CODE, MainActivity.this);

More about Input Parameters:
tippieId : id of tippie. If you are including offline tippie, it is the name of .tips file. e.g. for 1234567.tips file, 1234567 is the tippie id.
languagePack : it is two letter code for language pack. e.g. "en" for English, "hi" for Hindi,"mr" for Marathi. If your tippie is not langpack specific use "em" (Empty langpack).
tippieType : represents type of Tippie. currently we support only two formats e.g.TippieSDK.TYPE_DEFAULT or TippieSDK.TYPE_INTRO
isOnline : true if you are trying to load tippie from internet (i.e. our server).false if you have included .tips file in " assets/tippies/ " folder
context : Current context of the activity.
