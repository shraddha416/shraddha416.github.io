How to:

Download Library

Click on following button to download the library. This library is .aar file. You need to include this library inside your android project.

Generate API Key

Please sign up on our platform in order to register yourself as developer. After signing up, please create the profile for your application using following link.

If your Application already exists on our platform, then contact us to claim your application.


After your account has been created, you will be able to see API key in your account.

Configure Library

You need to configure this API key in your resources folder. Generally, you need to configure inside strings.xml file which is located in app / res / values / strings.xml 

                
Click To Copy

                
                  <?xml version="1.0"encoding="utf-8">
                  <resources>
                     <string name="api">cks2374ns3n2jb2b5h3b5353bk1bb52</string>
                  <resources>

After configuring API key, you can use our tutorial view in side your activity using our custom view.


              
Click To Copy

                
                    < tuteeasy.lib.triangular.view.TuteView
                        android:id="@+id/tuteview"
                        android:layout_width="wrap_content"
                        android:layout_height="match_parent"
                        android:api="@string/api"  />

Note: You need to set correct API key in android:apiotherwise tutorial will not show

Showing Tutorial List

You can use our Tutorial List api, to get list of tutorials for your account. Later, you can show this data using any type of list view.
